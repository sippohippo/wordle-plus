# Wordle+

> **Disclaimer: This is vibe coded slop.**

A Wordle clone that shows you the definition of every word you type — before you even submit it. Guesses and their definitions are tracked in a side panel.

## Features

- Standard Wordle gameplay (6 attempts, 5-letter word)
- Real-time dictionary lookup as you type (powered by [Free Dictionary API](https://dictionaryapi.dev/))
- Definition preview shown after the 5th letter, before you press Enter
- Side panel tracking all previous guesses with color-coded tiles and definitions
- Word validation via the dictionary API

## How to play

Open `wordle.html` in your browser. No server, no install, no build step.
